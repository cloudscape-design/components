# A5 — How do I build and run this project locally?

## Prompt

> How do I build and run this project locally? What commands should I use?

## Response

Here are the commands to build and run locally:

1. Install dependencies:

```sh
npm install
```

2. Build the project:

```sh
npm run build
```

3. Start the dev server (watches for changes + serves at http://localhost:8080):

```sh
npm start
```

For faster iteration, you can use `npm run quick-build` instead of `npm run build` — it skips documentation and screenshot testing builds, only compiling source code.

To run tests:

```sh
npm test          # all tests
npm run test:unit # unit tests only
npm run test:integ # integration tests
```

To lint:

```sh
npm run lint
```

Note: the project doesn't guarantee Windows support, so macOS (which you're on) is fine.
