# A3 — How do I test components in the browser?

Based on the project setup, you can test components in the browser using the dev server:

```sh
npm run start
```

This starts a watcher + dev server at `http://localhost:8080`. Development pages are served from `pages/<component-name>/` — so for example, to test the Button component, you'd navigate to `http://localhost:8080/pages/button/`.

If you need to test with React 18 specifically:

```sh
npm run start:react18
```
