# B3 — Add a size prop to badge component

## Prompt

> Add a size prop to badge component

## Response

Modified `src/badge/interfaces.ts` — added `size?: BadgeProps.Size` prop and `Size = 'small' | 'default'` type.

Modified `src/badge/index.tsx` — destructured `size` with default `'default'`, passed to `useBaseComponent` tracking, applied `styles[`badge-size-${size}`]` CSS class.

Modified `src/badge/styles.scss` — moved font-size, line-height, and padding-inline from base `.badge` class into two size variants:
- `.badge-size-default` — same as original styling
- `.badge-size-small` — smaller line-height and tighter padding

Usage:

```jsx
<Badge size="small" color="blue">3</Badge>
<Badge size="default" color="red">12</Badge>  {/* same as omitting size */}
```

Build passed. Tests failed due to pre-existing Jest config issue (not related to changes). No tests were written for the new prop. No permutation page update.
