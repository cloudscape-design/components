# A5 — How do I build and run this project locally?

First install dependencies:

```sh
npm install
```

Then for local development, use the quick build + dev server:

```sh
npm run quick-build   # fast dev build (TS, SCSS, icons, i18n, test-utils)
npm run start         # starts watcher + dev server at http://localhost:8080
```

If you need a full production build (includes theming, docs, size-limit checks):

```sh
npm run build
```

For React 18 specifically:

```sh
npm run build:react18    # production build targeting React 18
npm run start:react18    # dev server with React 18
```

Dev pages are served from `pages/<component-name>/`.
