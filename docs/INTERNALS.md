# Internal Utilities

Shared infrastructure lives in `src/internal/`. Always check there before introducing new utilities, hooks, or internal shared components.

Some shared utilities come from the [`@cloudscape-design/component-toolkit`](https://github.com/cloudscape-design/component-toolkit) package — hooks, utilities, and shared helpers used across Cloudscape repositories.

The [`@cloudscape-design/build-tools`](https://github.com/cloudscape-design/build-tools) package provides shared developer tooling.
